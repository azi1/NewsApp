# NewsApp

News App based on react native. For web support react-native-web is used.

# Setup

Run yarn or npm install

# Run Project

IOS: npm run ios
Android: npm run android
Web: npm run web 



# Note

Api's from newsapi.org not working in production mode on developer plan.
